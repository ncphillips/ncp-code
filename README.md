# ncphi vim 

## Keybindings
### Keys
* `C` : The `Ctrl` key
* `Leader` : The `,` key.

### Files
* **Save:** `,w`
* **Quit:** `,q`
* **Save & Quit:** `,wq`
* **Rename File:** `,n`

### Navigation

* **Page Forward:** `<C-b>`
* **Page Backward:** `<C-b>`

### File Browsing 

* **Toggle NERDTree:** `<C-n>`

### Typescript

* **Autocomplete:** `<C-x C-o>`
* **Go to def:** `<C-]>`
* **Go to use:** `<C-t>`
* **References:** `<C-^>`
* **Hint:** `,t`
* **Rename Symbol:** `,e`

### Vim Plugin System

* **Install Plugin:** `PlugInstall [name ...]`
* **Update Plugin:** `PlugUpdate [name ...]`
* **Upgrade vim-plug:** `PlugUpgrade`
